# java-web-buildr-blade-login

## Description
A POC for blade framework rendering html page intercepted with middleware.

## Tech stack
- cors
- middleware

## Docker stack
- vanto/apache-buildr:latest-jruby-jdk8

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
